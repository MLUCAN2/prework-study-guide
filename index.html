HTML

  • One of three cornerstone languages used in web development. Made up of elements that together provide content on the basic framework of all pages on the web.
  • Elements: can be seen on a page and divide/describe content to browsers and developers.
  • Elements inside of other element are considered child elements (i.e.h1, p, ul, etc.), with the element that holds them being the parent element.
  • Opening tag: used at the beginning of content
  • Closing tag: used at the end of content

Jave Script

  • A high level dynamic programming language that allows users to interact with websites. Takes a static HTML/CSS and makes it interactive.
  • Variable: a named container that can store data in our code.
  • Conditional Statements and Loops: control how code order is executed.
  • Arrays: a single variable used to hold a group of data.
  • "for" Loop: uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • "let" Variable: allows us to reassign the value of the variable over time. Unlike the "const" variable, "let" isn't static.
